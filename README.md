## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/SovaHandmade/Sova-Frontend
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run preview
   ```sh
   npm run dev
   ```
